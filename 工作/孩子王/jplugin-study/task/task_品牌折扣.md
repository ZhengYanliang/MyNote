##需求
```
品牌折扣的作用在于按品牌、品类批量给商品定折扣价。现有的价格促销仅支持根据单个商品设置规则。

品牌折扣的规则基本与现有的价格促销一致，在原有基础上增加以下改动：
1、折扣落到品牌、品类，支持品牌与品类的组合。
2、品牌，品类的维度均需增加供应商承担比例，供应商编码字段，非必填，都不填则同原“不补偿”。若只有承担比例，无供应商编码，则取erp默认的供应商。
3、本期品牌折扣仅针对线下，计算方式默认“交集”，并集对erp无效。

针对后台功能的改动：
1、选择门店处改为最终方案，即支持单个选择门店，分部。也支持批量添加门店编码；
2、销售渠道默认选择线下；
3、优惠类型仅显示并选择折扣；
4、增加“定义折扣品牌”项目，包括：交集（且），并集（或）
5、直接填写线下品牌，品类编码，以及对应的供应商和承担比例，比例填写正整数，供应商若为空，则表示取默认。都不填则视为不补偿。不支持批量导入。最多可增加20行。
6、原有的补偿方式模块去掉。

输入品牌，品类编码要自动带出来对应的名称

```
